# Luke's Collection of Custom UptimeKuma Themes

## Overview

This repository contains a curated collection of custom themes for [UptimeKuma](https://github.com/louislam/uptime-kuma), a self-hosted uptime monitoring tool.

## Inspiration

This project was initially inspired by [@dermv/cute-kuma](https://github.com/dermv/cute-kuma), which provided the foundation for exploring and creating beautiful theme customizations for UptimeKuma.

## Themes

A variety of theme options designed to enhance the visual appeal and usability of your UptimeKuma instance.
